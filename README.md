### Run Locally

To run **Crafti** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/crafti.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/crafti.git
```
